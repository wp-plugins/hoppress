== Description ==

This plugin will allow you to display a text link block with your affiliate ID on your site as a 
configurable widget. The widget can be added via the Appearance-->Widgets page in the WP-Admin are of your Wordpress site.

== Installation ==

1. Upload the hoppress.zip to your plugins directory and unzip it.
2. Login into your admin site and Activate the HopPress plugin. (Under Plugins)
3. Go to Appearance, Widgets
4. Select which sidebar you want to add the widget to.
5. Add the HopPress widget.
6. Then configure the settings, by clicking the Edit link.  Make sure you click Submit Changes.
